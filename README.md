# 商品管理システム

## 概要

このシステムでは、店舗で扱う商品の登録・編集等を行うことができます。

またTODOリスト及びカレンダー機能も使用することができます。


## 主な機能
- ログイン・ログアウト機能
- 商品一覧画面
- 商品新規登録・編集・削除機能
- 商品検索機能
- TODOリスト機能
- カレンダー機能


## 開発環境
- PHP7.4。21
- MYSQL14.14
- Laravel8.83.3
